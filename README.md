# 😷 Face Mask Detection System

A real-time face mask detection system built using deep learning and computer vision. The model classifies whether a person is wearing a mask or not using a CNN trained on a cleaned dataset.

---

## 🚀 Features

- Real-time face detection using webcam
- Mask vs No Mask classification
- Pre-trained CNN model included
- Cleaned and preprocessed dataset pipeline
- Lightweight and easy to run

---

## 🧠 Model Details

- Architecture: Convolutional Neural Network (CNN)
- Input: Face images (resized and normalized)
- Output: `Mask` / `No Mask`
- Framework: TensorFlow / Keras
